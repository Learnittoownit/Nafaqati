# 💰 Nafaqati

> Financial literacy for children through interactive family money management.

Nafaqati is an iOS application developed during the **Apple Developer Academy** to help Saudi children build healthy financial habits using the proven **3-Jar Method**: **Save, Spend, and Give**.

The application provides separate experiences for parents and children, allowing families to work together while encouraging children to learn responsible money management through real-life activities.

---

## ✨ Features

- 👨‍👩‍👧 Separate Parent and Child experiences
- 💰 3-Jar money management system (Save • Spend • Give)
- 🔐 Secure authentication
- ☁️ Real-time data synchronization
- 📊 Track savings and spending progress
- 👨‍👩‍👧 Parent monitoring and child permissions
- 📱 Modern SwiftUI interface
- 🧪 Tested with real families through TestFlight

## 🏗 Architecture

The application follows the **MVVM (Model–View–ViewModel)** architecture to ensure a clean separation between the UI, business logic, and data layer. Supabase powers authentication, real-time synchronization, and backend services.
